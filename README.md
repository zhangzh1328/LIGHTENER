# LIGHTENER
## Requirements
- python version : 3.8.17
- scanpy : 1.9.6
- sklearn version : 1.2.2
- torch version : 1.8.1
- torch-geometric version : 2.2.0

## Run LIGHTENER
We take Baron dataset as an example 
```bash
python main.py --data_path './Baron.h5' --save_model_path './save_file' --n_clusters 14
```
